# ObTools::G2D

Basic 2D graphics types. Header-only.

Part of the [ObTools](https://github.com/sandtreader/obtools) library collection.

## Quick Start

```cpp
#include "ot-g2d.h"
using namespace ObTools;

G2D::Point origin;           // (0.0, 0.0)
G2D::Point p(3.5, 7.2);

cout << "x=" << p.x << " y=" << p.y << endl;
```

## Build

```
NAME    = ot-g2d
TYPE    = headers
```
