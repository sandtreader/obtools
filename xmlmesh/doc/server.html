XMLMesh: The XMLMesh Server

The XMLMesh server is a standalone 'daemon' process which implements the core of the XMLMesh model.

This document describes the installation and configuration of the XMLMesh server.

Contents

Installing the XMLMesh server

The XMLMesh server is a single executable called 'xmlmesh-server'. Normally this will be installed to be started at boot time, and left running.
